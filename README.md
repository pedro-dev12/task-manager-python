# 📝 Task Manager - Python

Sistema de gerenciamento de tarefas desenvolvido em Python.

## 🚀 Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Salvamento automático em arquivo `.txt`

## 🛠 Tecnologias utilizadas

- Python 3
- Manipulação de arquivos
- Estruturas de repetição e condição

## ▶ Como executar

1. Clone o repositório:
   git clone https://github.com/pedro-dev12/task-manager-python.git

2. Acesse a pasta do projeto:
   cd task-manager-python

3. Execute o arquivo:
   python main.py
